常用的数组方法

  2. 下面方法将会以是否会改变原数组分类

    改变原数组

    splice() 添加/删除数组元素

    作用：通过删除或替换新的元素来修改原数组，该方法会修改原数组的指。该方法返回值是被删除的元素组成的数组

语法： array.splice(start,[deleteCount,item1,item2...])

参数说明：

  • start ： 指定修改的位置（从0计数）如果是负数，则表示从数组末尾开始。
  • deleteCount：表示要移除的元素个数
  • [item1,item2...]：要添加的数组元素，如果不指定则只删除数组元素。

实例

pop() 删除数组中的最后一个元素

作用： pop() 方法 将会删除数组中的最后一个元素，返回值是被删除的元素

语法： array.pop()

实例

shift() 删除数组中的第一个元素

作用：shift() 方法删除数组中的第一个元素，返回值被删除的元素

语法： array.shift()

实例

unshift() 在数组开头添加一个或多个元素

语法： array.unshift(element1[, element2....])

作用：unshift() 方法向数组的开头添加一个或多个元素，并返回数组添加后的长度

reverse() 将数组倒序

语法： array.reverse()

作用：reverse() 方法可以将数组的顺序颠倒 返回值是被颠倒后的数组

不改变原数组

join() 将数组按照指定分隔符转成字符串

作用：将 数组元素按照指定的分隔符转成字符串，默认是按照 逗号 分隔。

语法： array.join([*])

实例

toString() 将数组转成字符串 不推荐

语法：array.toString()

作用： 将数组转成以逗号分隔的字符串，因 toString 只能将数组转成以逗号分隔的字符串不能自定义所以不推荐使用，不如使用 join() 来转换。比起起使用 toString() 方法，当 数组 和 字符串相连接时数组会转成 以逗号分隔的字符串，这也是不推荐的使用 toString() 方法的原因之一。

实例

concat() 合并两个数组或多个

作用： 方法合并两个或多个数组，并返回一个新的数组

语法： array.concat(array1[,array2,array3])

参数说明： array 和 array1…. 代表要合并的数组

实例

indexOf() 查找数组是否含有某元素

作用： indexOf() 可以查找数组中是否含有某个元素，有：返回改元素在数组中的下标，无：返回 -1。

语法：array.indexOf(element[,fromIndex])

参数说明： element 代表查找的元素 fromIndex 从哪个索引开始查找

实例

lastIndexOf 查找数组是否包含某元素

作用 ： 和 indexOf 作用一致用法一致，与之不同的是 lastIndexOf 是倒序查找元素。

includes() 查找数组是否包含某元素 返回布尔

作用： includes() 查找数组是否包含某元素 返回布尔

语法：array.includes(element)

参数说明： element 查找的元素

实例

slice()

作用：浅拷贝数组。slice()方法返回一个新的数组对象，这个数组对象由 start 和 end 决定元素的浅拷贝 (包括 satrt 不包括 end)

实例：

参数：
start: 提取元素的起始索引，果如出现 slice(-2) 代表从数组的 倒数第二个元素开始提取

end：提取元素的终止索引

sort()

作用：对数组元素进行排序，并返回数组。默认排序顺序是将元素转为字符串然后比较它们的UTF-16代码单元值

实例 ：

语法： arr.sort([compareFunction])

参数：

  1. compareFunction： 可选参数 用来指定按照某种顺序进行排列的函数，如果省略，元素按照转换为的字符串的各个字符的Unicode位点进行排序。
  2. 1 firstEl: 第一个比较的元素
  3. 2 secondEl: 第二个比较的元素

返回值：排序后的数组，已经原地排序了。改变了原数组。

描述：
如果指明了 compareFunction(),那么数组会按照调用该函数的返回值排序。

  • 如果要比较的是数字而非字符串 compareFunction(a, b) {return a - b} 函数会对数组进行升序
    常用例子： 也可以按照某个属性进行排序

flat()

语法： array.flat([depath])

参数：

depath 可选参数，指定提取嵌套数组的结构深度，默认值为 1

返回值： 一个包含数组与子数组的新数组
