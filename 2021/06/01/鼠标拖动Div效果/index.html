本次 demo 用到一些属性

elment.clientWidth 获取元素的宽度 包括自身的 padding 内容的宽度 不包括边框、返回值不带单位
elment.clientHeight 获取元素的高度 包括自身的 padding 内容的宽度 不包括边框、返回值不带单位

element.offsetTop elment 必须是相对定位的元素 该 API 返回 元素距离父元素上方的距离
element.offsetLeft elment 必须是相对定位的元素 该 API 返回 元素距离父元素左方的距离
